
A React-based app that allows users to search for movies using the OMDB API.

## Features
- Search movies by title
- View full movie details
- Filter by type (movie, series, episode)
- Pagination support
- Add to favorites (localStorage)
- Clean UI with Tailwind CSS

## Tech Stack
- React
- React Router
- Axios
- Tailwind CSS
